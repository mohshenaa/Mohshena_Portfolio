# 🌐 Mohshena Akter Meem – Portfolio

This is my personal portfolio website built using ASP.NET Core MVC to showcase my projects, skills, and development journey.
It includes a fully functional Admin Dashboard to manage content dynamically.

# 🚀 Features
✨ Modern and responsive UI
🌙 Dark / Light mode toggle
⌨️ Typewriter effect in Hero section
🎯 Smooth scroll animations (About, Skills, Projects)
📊 Admin Dashboard with real-time data
🧩 Dynamic Projects & Skills management
📩 Contact form with message storage
🔐 Role-based access (Admin only controls)
🧑‍💻 Tech Stack
# 🔹 Backend
ASP.NET Core MVC
C#
Entity Framework Core
# 🔹 Frontend
HTML5
CSS3 (Responsive + Animations)
Razor Views
# 🔹 Database
SQL Server

# 🛠️ Installation & Setup
Clone the repository:
git clone https://github.com/mohshenaa/your-repo-name.git
Navigate to project folder:
cd your-repo-name
Restore dependencies:
dotnet restore
Apply database migrations:
dotnet ef database update
Run the project:
dotnet run
Open in browser:
https://localhost:5001
